# 🔗 Vorazone Connect Page

Haiii! Selamat datang di project **Vorazone Connect Page** 👋
Halaman ini adalah **link hub sederhana** yang dibuat untuk brand dropshipping saya **Vorazone**, sebuah toko yang fokus pada **smart gadgets dan elektronik modern**.

Halaman ini berfungsi sebagai **pusat koneksi** untuk mengarahkan pengunjung ke berbagai platform Vorazone seperti media sosial, marketplace, dan halaman lainnya.

---

## 🛠 Tech Stack

Framework: Next.js
Library: React
Styling: Tailwind CSS
Animation: Framer Motion
Icons: Lucide

Deployment & Infrastructure:
CDN & PaaS: Netlify
Static Site Generator: Next.js
Web Server: Next.js

---

## 💡 Inspirasi

Project ini terinspirasi dari konsep **Link-in-bio pages** seperti Linktree, di mana semua link penting dikumpulkan dalam satu halaman yang sederhana dan mudah diakses.

Namun halaman ini dibuat sebagai **custom connect page khusus untuk brand Vorazone**.

---

## 📱 Tujuan Project

Project ini dibuat untuk:

* Menyediakan **halaman koneksi brand**
* Mengumpulkan semua **link penting Vorazone**
* Membuat **landing page sederhana dan cepat**
* Mempermudah akses pengunjung dari media sosial
